# 🚀 React Native + NativeWind Starter

A clean, ready-to-use boilerplate for building modern React Native applications. I am using this template as a foundational starting point to explore the React Native ecosystem and bring Tailwind CSS styling to mobile development.

## 🛠 Tech Stack

* **Framework:** React Native & Expo
* **Styling:** NativeWind (Tailwind CSS)
* **Language:** TypeScript
* **Routing:** Expo Router

## ⚡ Quick Start

To run this template locally:
   **Install dependencies:**
   ```bash
   npm install
   npx expo start


```

Or Native way
```bash
npx create-expo-app newApp --template https://github.com/Aravind-tanneri/ExpoApp-Template
cd newApp
npx expo start -c
